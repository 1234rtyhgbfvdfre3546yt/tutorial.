Como mudar os textos

A maioria dos textos do site você muda sem precisar de código nenhum. Mas nos widgets (Tracker, Meal Plan etc.) os textos ficam dentro de um arquivo de código.

1
Texto normal no Google Sites
Títulos, parágrafos, legendas das imagens
▾

Isso é o mais fácil do mundo. É igual ao Word:

Abra o Google Sites — entre no site e clique no ícone de lápis ✏️ no canto superior direito para entrar no modo de edição.
Clique duas vezes — em qualquer texto que você quer mudar. Vai aparecer um cursor de digitação igual ao do Word.
Edite e salve — mude o texto, depois clique em "Publicar" no canto superior direito para salvar.

A regra de ouro: texto fica entre as setas > e <

No código HTML, o que você vê na tela sempre fica entre o fechamento de uma tag (>) e a abertura da próxima (<). Veja o exemplo abaixo e mude o texto — a prévia ao lado atualiza na hora:

Código — mude o texto entre as setas
Como fica — atualiza enquanto você digita
👆 O texto que está escrito entre > e < é exatamente o que aparece no resultado. O resto (as partes com igual, aspas e ponto e vírgula) é só a aparência — não mexa nisso se não sabe o que é.

Para achar um texto específico no código:

Abra o código — no Google Sites, clique no widget, depois nos três pontinhos ⋮ e em "Editar".
Use Ctrl+A — seleciona todo o código. Depois Ctrl+C para copiar e cole em um bloco de notas.
Use Ctrl+F — abre a busca. Digite o texto que quer achar (ex: "Push-Up") e ele vai destacar onde está.
Mude só o texto — substitua o que quer trocar, sem apagar nenhuma aspa, chave ou ponto e vírgula ao redor.
Copie de volta — selecione tudo, copie e cole de volta no campo de código do Google Sites.
⚠️
Não mexa no que não é texto

Se você ver um sinal de : # seguido de letras e números (ex: #c8ff00), isso é uma cor. Se ver px ou rem, é um tamanho. Só mude os textos que você reconhece como palavras normais.

🎨 Parte 2

Como mudar as cores dos widgets

As cores dos widgets estão concentradas em um único lugar no começo do código, chamado de "variáveis". Você muda o valor lá e a cor muda em todo o widget de uma vez.

🎨
O que é um código de cor?

As cores no HTML são escritas como #c8ff00 — isso se chama "hex". Você não precisa memorizar esses códigos: basta ir no Google e digitar "color picker", escolher a cor que quer, e copiar o código que começa com #.

Resultado não ficou bom? Use um destes ajustes:

🔧 Ajuste 1 — Ficou com muitos detalhes
Simplifique mais. Use apenas 3 a 4 tons de cor por região, sem textura ou sombreado complexo. O resultado deve parecer uma ilustração de app, não uma foto realista.

Editando os widgets interativos

Cada widget é um arquivo de código HTML que foi colado dentro do Google Sites. Para editar, você abre esse código, faz a mudança e cola de volta. Não precisa entender o código — só precisa saber onde procurar.

📌
Como abrir o código de um widget no Google Sites
  • Entre no modo de edição (ícone de lápis ✏️)
  • Clique no widget que quer editar
  • Clique nos três pontinhos ⋮ que aparecem
  • Clique em "Editar"
  • O código vai aparecer numa caixa de texto — edite ali mesmo

No código do Meal Plan, cada refeição tem um bloco chamado recipes:. Dentro dele, cada receita tem esses campos que você pode mudar:

Campo no códigoO que aparece no site

💡
Os macros femininos são automáticos

Você só precisa preencher os valores masculinos (m: { p, l, c, k }). O código já calcula os femininos automaticamente como 85% dos valores masculinos — nada para mudar.

⚠️
Atenção com as aspas e vírgulas

Cada ingrediente fica dentro de { qty: '...', item: '...' } e cada passo fica entre aspas simples '...'. Se você apagar uma vírgula ou aspa, o widget quebra. Use Ctrl+Z se isso acontecer.

3
Mapa Financeiro — adicionar apps e canais
Novos "nós" no mapa interativo
▾

O mapa tem "nós" (as bolhas) e "linhas" que os conectam. Para adicionar um novo app ou canal, você precisa fazer duas coisas:

Adicionar o nó novo — dentro do array nodes, copie um bloco type: 'leaf' existente e cole logo abaixo, mudando o id, as posições x e y, e o conteúdo HTML.
Adicionar a linha de conexão — dentro do array lines, adicione uma nova linha no formato { from: 'cat-apps', to: 'id-do-novo-no', cls: 'conn-apps' }. Use cat-yt e conn-yt para canais do YouTube.
⚠️
As posições x e y no mapa

O centro do mapa fica em cx, cy (metade da tela). Posições negativas em x vão para a esquerda (apps), positivas vão para a direita (YouTube). Ajuste o número até ficar bem posicionado.
